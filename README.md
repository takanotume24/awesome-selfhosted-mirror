# Awesome Self-Hosted Mirrors

A list of open-source tools for hosting and managing local mirrors of popular software repositories. These solutions help reduce external bandwidth usage, speed up installations and updates, and provide offline access for environments without full internet connectivity.

---

## APT Mirrors

- [aptly-dev/aptly](https://github.com/aptly-dev/aptly)  
- [apt-mirror/apt-mirror](https://github.com/apt-mirror/apt-mirror)  
- [rickysarraf/apt-offline](https://github.com/rickysarraf/apt-offline)  
- [Gargravarr2112/mirror-rsync](https://github.com/Gargravarr2112/mirror-rsync)  

## Cargo Mirrors

- [panamax-rs/panamax](https://github.com/panamax-rs/panamax)  

## PyPI Mirrors

- [devpi/devpi](https://github.com/devpi/devpi)
